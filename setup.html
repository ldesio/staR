How to run staR

  • On Windows:
    • Download and install R (version 3.5.0 required). More info here.
    • Download and install Pandoc (more info here):
      • Windows 64-bit (use as default)
      • Windows 32-bit
    • Perform the installation of required R packages by running win\staR_setup.bat.
    • Run win\staR.bat.
  • On Mac OS X:
    • Download and install R (version 3.5.0 required). More info here.
    • Download and install Pandoc (latest version here). More info here.
    • Perform the installation of required R packages by running the staR_setup script in this folder:
      • Ctrl-click on the mac/staR_setup file (this is needed, otherwise your Mac won't let you run the file for security reasons);
      • Click on "Open";
      • Confirm that you wish to open and run the file;
      • A text window will popup, showing various setup messages; wait until a "[process completed]" message, then close this window.
    • Run staR in this folder:
      • Ctrl-click on mac/staR file (this is needed, otherwise - the first time - your Mac won't let you run the file for security reasons);
      • Click on "Open";
      • Confirm that you wish to open and run the file;
      • A text window will popup, showing various messages; wait until a Safari window pops up with staR.
      • If you don't see any staR window, go on Safari and navigate to http://localhost:8000
  • After staR has started, check that you can perform a basic analysis:
    • In the "command" box, type use http://bit.ly/wvs05es and hit Enter
    • you should see a message reporting that the dataset has been successfuly opened. You are now ready.
  • Enjoy ;)
